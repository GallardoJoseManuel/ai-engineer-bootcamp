# Dataset Analyzer

## Description

Dataset Analyzer is a simple Python project built with NumPy to practice working with multidimensional arrays and basic statistical analysis.

The program generates a random dataset and calculates useful information about it.

## Features

- Generate random datasets
- Display dataset shape
- Display data type
- Display memory usage
- Calculate column statistics
- Calculate row statistics
- Mean
- Maximum
- Minimum
- Standard deviation

## Technologies

- Python 3
- NumPy
