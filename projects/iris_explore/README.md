# Iris Explore

## Description

This project is an introduction to data analysis using the famous Iris dataset and the Pandas library.

The goal is to explore the dataset, inspect its structure, compute descriptive statistics and perform basic filtering and grouping operations.

## Features

- Load the Iris dataset from a CSV file.
- Inspect the DataFrame structure.
- Display descriptive statistics.
- Select one or multiple columns.
- Filter rows using boolean masks.
- Group data by flower species and compute summary statistics.

## Technologies

- Python 3
- Pandas
- NumPy

## Concepts Practiced

- DataFrame
- Series
- CSV loading
- Data inspection
- Boolean indexing
- Filtering
- GroupBy
- Descriptive statistics

## How to Run

```bash
python main.py
```

During this project I learned:

- The difference between a DataFrame and a Series.
- How to inspect a dataset using `shape`, `columns`, `dtypes`, `info()` and `describe()`.
- How boolean masks work in Pandas.
- How to filter rows based on conditions.
- How `groupby()` can summarize data by categories.
